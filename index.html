Problem

こんな状態になっていませんか？

施工管理の仕事は、人が判断すべきことが多い一方、書類・整理・共有に時間を取られやすい仕事でもあります。

01

日報や週報に毎日追われる

現場が終わってから報告書づくりに時間がかかり、残業が増えやすい。

02

施工写真の整理が大変

写真の選別、並び替え、説明文づけに手間がかかり、担当者依存になりやすい。

03

定例資料が毎回ゼロから

会議資料のたたき台がなく、似た説明を何度も作り直している。

04

生成AIに興味はあるが不安

何から始めればいいか分からず、現場で本当に使えるのか判断できない。

Solution

この研修は、施工管理の“書類仕事”から始めます

最初から難しい仕組みや開発の話には入りません。明日から現場で使える順に、3段階で進めます。

Step 1

まずは生成AIの基本

ChatGPT、Gemini、Copilotなどを想定し、安心して使うための考え方と、指示の出し方の型を学びます。

Step 2

施工管理の実務で演習

日報、写真整理、定例資料、安全書類、品質記録を題材に、実務に近い形でテンプレートを作ります。

Step 3

必要な会社だけ Claude Code

発展編では Claude Code を使い、ファイル整理、差分確認、資料の下書きなどを一歩進めて扱います。

Claude Code

Claude Code は“発展編”として位置づけます

Claude Code を最初から全員に教えるのではなく、まずは生成AIの基本活用を揃え、そのあと必要な担当者にだけ広げます。

こう伝えると分かりやすいです

Claude Code は、パソコンの中のファイル整理や、資料の下書き作成を手伝う生成AIツールのひとつです。 施工管理向け研修では、難しい開発学習ではなく、業務で使う範囲に絞って扱います。

発展編で扱うこと

  • ファイル整理ルールの支援
  • 会議資料や説明文の下書き
  • 前回資料との差分確認
  • 繰り返し作業の型づくり

FAQ

よくある質問

本当に非IT人材でも受講できますか？

はい。生成AI活用の基礎ワークショップは、もともとビジネス職・非エンジニア向けの設計です。施工管理向け版でも、その前提を維持しています。

Claude Code は必須ですか？

必須ではありません。まずは一般的な生成AIの活用から始め、必要な場合だけ発展編として Claude Code を追加します。

どの業務から始めるのがよいですか？

日報、写真整理、定例資料の3つから始めるのが最も分かりやすく、社内で成果を共有しやすいです。

オンラインでも実施できますか？

はい。オンライン・対面の両方に対応しています。受講人数や環境に合わせて進め方を調整します。

自社の帳票に合わせた内容にできますか？

可能です。2Day実践版や伴走版では、事前ヒアリングのうえで帳票や業務フローに合わせてカスタマイズします。
